# Ola-Dashboard

# Project Overview

This project is a complete Power BI analytics dashboard built to analyze OLA ride bookings, cancellations, revenue patterns, and customer experience over a selected time period.
It provides a data-driven view of how OLA is performing operationally and financially.

The dashboard is fully interactive, allowing users to adjust time filters and navigate through different report sections.

# Objectives of the Dashboard

This dashboard was created to:

Track total bookings and booking value over time

Understand booking outcomes (success / canceled by driver / canceled by customer / driver not found)

Monitor ride volume trends

Evaluate performance metrics across:

Vehicle Types

Revenue

Cancellations


# Tools & Technologies Used

Power BI Desktop

DAX for calculated fields

Power Query for data cleaning

Excel / CSV dataset (Booking data)

GitHub for version control


# Insights You Can Derive

Percentage of successful rides

Driver vs customer cancellation behavior

Total revenue generated

Daily ride demand patterns

Performance comparison across vehicle categories

Ratings

Provide a simple, business-friendly way to interpret large booking datasets
